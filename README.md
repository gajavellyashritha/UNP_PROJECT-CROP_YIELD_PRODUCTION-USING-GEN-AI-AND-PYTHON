# UNP_PROJECT-CROP_YIELD_PRODUCTION-USING-GEN-AI-AND-PYTHON

This project predicts crop yield using a regression-based machine learning model. It uses agricultural data like state, district, crop type, and season to analyze patterns and generate accurate predictions. Data preprocessing includes cleaning, handling missing values, and encoding categorical features. The model is evaluated using R², MAE, and MSE to ensure accuracy. This system helps in data-driven decision-making for crop planning, resource optimization, and improving agricultural productivity.
